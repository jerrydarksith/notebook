# Personal Information System

Персональна система збереження інформації. Telegram буде єдиним інтерфейсом
взаємодії, але бізнес-логіка не залежатиме від Telegram.

## Поточний етап

Підготовлено базову структуру проєкту та початкову SQLite-схему. Telegram
авторизація, заявки на доступ, меню та категорії будуть додані наступними
окремими етапами.

## Вимоги

- Python 3.14;
- `python-telegram-bot`;
- `python-dotenv`;
- SQLite зі стандартного модуля Python.

## Конфігурація

1. Скопіюйте `.env.example` у `.env`.
2. За потреби змініть `DATABASE_PATH`.
3. Встановіть залежності, визначені у `pyproject.toml`.
4. Запустіть застосунок:

   ```bash
   python3 -m personal_bot
   ```

Після запуску створюється файл SQLite та початкові таблиці.

Деталі архітектури наведено в [ARCHITECTURE.md](ARCHITECTURE.md).
